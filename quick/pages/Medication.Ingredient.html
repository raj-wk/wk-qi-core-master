Medication.Ingredient

  • Active or inactive ingredient

    Identifies a particular constituent of interest in the product.

    • Key:  = Must support,  = QICore-defined extension,  = Is-Modifier

       Field Card. Type and Description
        amount 0..1 Ratio
      Specifies how many (or how much) of the items there are in this Medication. For example, 250 mg per tablet. This is expressed as a ratio where the numerator is 250mg and the denominator is 1 tablet.
        id 0..1 String
      unique id for the element within a resource (for internal references). This may be any string value that does not contain spaces.
        isActive 0..1 Boolean
      Indication of whether this&nbsp;ingredient affects&nbsp;the therapeutic action of the drug.
        item[x] 1..1 Reference
      The actual ingredient - either a substance (simple ingredient) or another medication.
